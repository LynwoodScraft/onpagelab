## What the report contains

### Global SEO Score (0–100)
An aggregated score calculated across all analysis axes. A score ≥ 80 indicates a well-optimized page. Below 60, corrective actions are needed to progress in the SERPs.

### Sub-scores by axis
The report details 5 independent sub-scores:
- **Technical**: HTTP status, title, meta description, canonical, robots, Open Graph
- **Semantic**: keyword density in content, headings, title and alt attributes
- **Structure**: H1–H6 hierarchy, H1 uniqueness, keyword in headings
- **Internal linking**: link count and quality, anchor texts, broken links
- **Tags**: pixel length of title and meta, social tag presence

### Detected issues list
Issues are ranked by priority: **Critical** (blocking), **Important** (high impact), **Advisory** (optimization). Each issue comes with an explanation and a concrete recommendation.

## The 12 on-page factors analyzed

1. **Title tag** — Presence, pixel length (optimal: 500–590 px), keyword at beginning
2. **Meta description** — Presence, pixel length (optimal: 430–920 px), call to action
3. **H1 tag** — Presence, uniqueness, length, keyword inclusion
4. **H2–H6 structure** — Logical hierarchy, level skips, keyword in H2s
5. **Keyword density** — Occurrence rate in body, headings, title and alt
6. **Content length** — Word count vs. competing pages
7. **Internal links** — Count, anchor texts, target URL quality
8. **External links** — Count, attributes, linked domains
9. **Images** — Alt attributes, estimated compression, lazy loading
10. **Canonical tag** — Self-referencing, external, or absent
11. **Robots tag** — Index/noindex, follow/nofollow
12. **Open Graph & Twitter Card** — Essential social tags presence

## How to use the SEO analyzer

### Step 1: Identify the page to optimize
Start with strategic pages: those targeting high-volume keywords, close to the top 10, or generating organic traffic without converting.

### Step 2: Define your target keyword
Each page should target a unique primary keyword. Avoid cannibalization by ensuring only one page on your site targets the same term.

### Step 3: Run the analysis
Paste the URL and keyword into the form. The report is ready in 30 seconds.

### Step 4: Fix critical issues first
Start with "Critical" issues (missing title, missing H1, accidental noindex). These can prevent page ranking regardless of content quality.

### Step 5: Optimize iteratively
After each change, run a new analysis to measure impact. A score above 80/100 indicates a well-optimized page.

## FAQ

**Is the tool really free?**
Yes, basic analysis is 100% free with no account required. Enter a URL and get your report instantly.

**Does the keyword field have to be filled in?**
No. Without a keyword, OnPageLab performs a complete technical audit. Adding a keyword activates semantic analysis (density, heading presence, semantic score).

**Can it analyze JavaScript-rendered pages (SPA, React, Next.js)?**
OnPageLab crawls the HTML returned by the server. For client-side rendered sites, dynamically loaded elements may not be included. SSR or SSG sites will yield the most complete results.

**How soon will I see improvements after optimization?**
Google's re-indexation delay varies from a few hours to several weeks depending on crawl frequency. To speed things up, submit the URL in Google Search Console after making changes.
