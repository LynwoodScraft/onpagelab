## The 10 technical elements audited

OnPageLab systematically checks: HTTP response code, title tag (presence, pixel length, keyword), meta description (presence, pixel length), H1 (presence, uniqueness, keyword), canonical tag (self-referencing, external, absent), robots meta tag (noindex/nofollow detection), Open Graph (og:title, og:description, og:image, og:url), Twitter Card, lang attribute, and viewport meta tag.

## Common technical issues detected

- Title missing or too long → critical ranking impact
- Meta description absent → loss of SERP control
- Accidental noindex → page not indexed, all SEO effort wasted
- Duplicate H1 → ambiguous primary topic signal
- Wrong canonical tag → page devalued in favor of another URL

## FAQ

**What's the difference between on-page technical SEO and site-wide technical SEO?**
Site-wide technical SEO covers the whole domain (robots.txt, sitemap, site speed, URL structure). On-page technical SEO focuses on signals present inside a specific page. OnPageLab specializes in this second dimension.

**Can OnPageLab detect an accidental noindex?**
Yes. If the analyzed page contains `<meta name="robots" content="noindex">` or an HTTP `X-Robots-Tag: noindex` header, OnPageLab flags it as a critical issue with immediate recommendations.
